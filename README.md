# candy-crush
Candy crush en C avec ncurses

## Comment jouer 
Vous devez déplacer le curseur avec les flèches directionnels.

Pour échanger deux bonbons appuyez sur Entrée et sélectionner la direction dans laquelle vous voulez échanger les bonbons.

Si vous alignez 3 bonbons ou plus vous les faites éclater (vous gagnez des points) !
